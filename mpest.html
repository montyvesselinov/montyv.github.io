MPEST / MPRUN

A parallel version of the code PEST (Doherty 2009) called MPEST has been developed to optimize the solving of parallel optimization problems at the LANL multi-processor clusters.

MPEST has been applied in many parallel computing projects worldwide.

The parallelization in the MPEST framework is performed using the code MPRUN, a code that has been also developed at LANL.

MPRUN is using POSIX threads to fork individual model runs to processors; this approach has been demonstrated to be very effective and superior to using MPI calls in terms of computational performance.

Currently, the parallel subroutines developed for MPEST/MPRUN are imported and further developed in the code MADS.
